FAQ

How do I deploy this site?
  1. Save this file as index.html in your GitHub repo.
  2. In the repo settings, enable GitHub Pages → Source: main, folder: /root.
  3. Open the published URL. Done.
How do I hook up email & SMS?

Connect the form submit to your stack using a serverless function or tools like Zapier/Make. See inline comments in the code for /api/waitlist ideas.

Is my data private?

Yes. Spirit is designed consent‑first. We collect only what’s needed to deliver your experience and you can delete it at any time.
